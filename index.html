5. Jak skonfigurować system Git w taki sposób, aby po wykonaniu zatwierdzenia automatycznie uaktualniał centralne repozytorium?

Najprościej ustawić aby instrukcja git push wykonywała się automatycznie po każdym wykonanym zatwierdzeniu. W tym celu należy ustawić odpowiedni skrypt w katalogu .git/hooks. W tym przypadku skrypt będzie się nazywał post-commit i będzie zawierał polecenie git push. Można to otrzymać poleceniem: 

echo 'git push' > .git/hooks/post-commit
Zobacz: link

▶

6. Jak wyeksportować zawartość dowolnego zatwierdzenia albo galęzi do dowolnie podanego katalogu na dysku?

Można skorzystać z polecenia git archive i narządzia tar: 

git archive master | tar -x -C /sciezka/do/katalogu
